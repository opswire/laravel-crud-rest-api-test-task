## Инструкция По Запуску:


+ ``` Clone the repo```


+ ``` cd laravel-crud-rest-api-test-task```


+ ```cp .env.example .env```


+ ```docker-compose up --build -d```


+ ```docker-compose run --rm -it app composer install```


+ ```docker-compose run --rm -it app php artisan migrate --seed```
